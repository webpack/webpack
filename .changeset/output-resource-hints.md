Add `output.resourceHints` for resource-hint (`<link rel="preload/prefetch/modulepreload/preconnect">`) emission. It accepts the initial dependency-graph shorthand (`boolean | "prefetch" | "preload" | "none" | HtmlResourceHint[] | function` — equivalent to `{ initial: … }`) or the object form `{ initial, urlHints, preconnect, modulePreloadPolyfill, manifest }`. The callback receives the auto `defaultHints` (each carrying `hostChunks`) plus `{ entryName, entrypoint, hostType, compilation }`; `manifest` writes an SSR JSON file; `preconnect` warms a cross-origin `output.publicPath` origin; `modulePreloadPolyfill: false` opts out of the inline polyfill. Hints emit into the extracted HTML `<head>` for HTML entries and via a chunk-startup JS runtime otherwise, and `stats.entrypoints[name].resourceHints` exposes the resolved list. Also adds `module.parser.<type>.urlHints` rule sets (with a project-wide `output.resourceHints.urlHints` shorthand), `module.parser.css.fontPreload`, and `module.parser.javascript.dynamicImportCssPreload`.
